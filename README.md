# Senior Ease

Aplicacao para gerenciamento de tarefas para público senior baseada em Next.js com App Router.

## Tecnologias

Este projeto utiliza:

- Next.js App Router
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Rodando localmente

Requisitos:

- Node.js
- npm

Passos:

```sh
cd frontend
npm install
npm run dev
```

Depois, acesse http://localhost:3000 no navegador.
